个人博客搭建

前言

最近本科阶段刚结束，迎来了大量的空闲时间，所以就很想搞事情…。于是接下来几天用python写了个简单爬虫程序后高高兴兴的想去博客记录收获，然后…然后…要不再搭建个个人博客（@-@）？？？又折腾了三天终于初步完成了，以下将列出详细过程。

环境配置

以下的所有配置均是在macOS下实现，但对于window也均有较大的参考价值。

安装Node.js

直接到官网下载安装文件，安装即可。

安装git

在终端用如下命令安装：

sudo apt-get install git

安装hexo

终端执行以下命令：

$ sudo npm install -g hexo

输入mac的用户密码。

注意：mac终端输入密码处显示🔑图标，不会实时显示所输密码，只需确保输入后回车即可

安装完成后进入本地博客的根目录：

cd NTGN.github.io

如果找不到路径可以到终端查看安装时返回的信息，此外NTGN.github.io为本地博客的文件名

GitHub 配置

以上得到的个人博客静态页面的访问链接是本机地址，理应再配上一个独有的域名，由于目前刚毕业又没入学，失去了学生身份，享受不到腾讯云价格低廉的服务。所以，最终我只将其和GitHub绑定，后续从GitHub再到个人域名的配置就没再进行。

建仓

首先需在GitHub上建仓，操作简单不细说了，但有个重要细节需注意，名称应与账号名保持一致（不太懂），不然可能导致失败。例如用户名为yyy，则仓名为yyy.github.io。

也可以先建仓再根据仓名去进行一开始的环境配置

发表博客

新建博客

其实发布一篇博客很简单。首先：

hero new post "第一篇博客"

根据终端返回的信息可知在根目录source文件夹下创建了一个_post文件夹，他是用来存放我们的博客文章的，是markdown类型。
